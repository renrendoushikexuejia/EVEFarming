# EVEFarming


脚本更新至2023年6月6日

EVE游戏刷官员装备的脚本，使用毒蜥装备无人机。出现官员之后会声音提醒和微信消息提醒。

游戏内要先使用MyEVESetting更改游戏界面

搭配radmin使用，在远程电脑上运行游戏和radmin。

在主控端运行脚本，通过radmin控制游戏。主控端可以控制多个radmin窗口，一般选择两个，以保证每个窗口都能及时操作。
